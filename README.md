# HOW TO RUN THE PROJECT

### STEP 1: 
 - Install Node.js
 - Use this command to install project dependencies:
  ```
  npm install
  ```
### Step 2: 
 - Execute the following command in project root folder to run task 2 solution:
  ```
   nodemon
  ``` 
### Step 3: 
 - Open the project in browser: http://localhost:8000/
